HTML Form Elements

What it is
- HTML provides different elements to collect user input inside a `<form>`.

Common form elements
- `<input>` - input control (text, number, etc.)
- `<label>` - label for an input
- `<select>` - drop-down list
- `<option>` - option inside a select
- `<textarea>` - multi-line text input
- `<button>` - clickable button
- `<fieldset>` and `<legend>` - group related fields
- `<datalist>` - list of suggested options for an input
- `<output>` - result of a calculation

Best example
```
<form action="/action_page.php">
  <label for="cars">Choose a car:</label>
  <select id="cars" name="cars">
    <option value="volvo">Volvo</option>
    <option value="saab">Saab</option>
    <option value="fiat" selected>Fiat</option>
    <option value="audi">Audi</option>
  </select>

  <label for="message">Message:</label>
  <textarea id="message" name="message" rows="4" cols="30">
The cat was playing in the garden.
  </textarea>

  <input list="browsers" name="browser">
  <datalist id="browsers">
    <option value="Edge">
    <option value="Firefox">
    <option value="Chrome">
    <option value="Opera">
    <option value="Safari">
  </datalist>

  <button type="button" onclick="alert('Hello World!')">Click Me!</button>
  <input type="submit" value="Submit">
</form>
```

Notes
- Use `<label>` with `for` to improve accessibility.
- Always set the `type` attribute on `<button>`.
