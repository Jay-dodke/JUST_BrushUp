HTML Background Images

What it is
- A background image can be set on almost any HTML element using CSS `background-image`.
- You can apply it inline with the `style` attribute or in a `<style>` block.

Background on an element (inline)
```
<p style="background-image: url('oldman.jpg');">
  This paragraph has a background image.
</p>
```

Background on a page (CSS)
```
<style>
  body { background-image: url('nature.jpg'); }
</style>
```

Repeat and cover
```
<style>
  body {
    background-image: url('smileFace.jpg');
    background-repeat: no-repeat;
    background-attachment: fixed;
    background-size: cover;
  }
</style>
```

Best practice
- Use `background-repeat: no-repeat` if you do not want tiling.
- Use `background-size: cover` to fill the area while keeping proportions.
