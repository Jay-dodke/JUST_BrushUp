HTML Images

What it is
- Images are added with the `<img>` tag.
- `<img>` is empty (no closing tag) and uses attributes.
- Two required attributes: `src` (image path) and `alt` (alternate text).

Basic syntax
```
<img src="url" alt="description">
```

Why `alt` matters
- Shows text if the image fails to load.
- Helps screen readers describe the image.

Size example
```
<img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">
```

Best practice
- Always include meaningful `alt` text.
- Use correct paths so images don’t break.
- Set width/height to reduce layout shift.

Example with image link
```
<a href="default.asp">
  <img src="smiley.gif" alt="HTML tutorial" style="width:42px;height:42px;">
</a>
```
