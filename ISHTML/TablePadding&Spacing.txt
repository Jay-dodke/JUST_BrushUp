HTML Table Padding & Spacing

What it is
- Cell padding is the space between a cell’s edge and its content.
- Cell spacing is the space between table cells.
- Padding is set with CSS on `th` and `td`.
- Spacing is set with CSS `border-spacing` on the `table`.

Cell padding example
```
th, td {
  padding: 15px;
}
```

Different padding sides
```
th, td {
  padding-top: 10px;
  padding-bottom: 20px;
  padding-left: 30px;
  padding-right: 40px;
}
```

Cell spacing example
```
table {
  border-spacing: 30px;
}
```

Notes
- Default padding is 0.
- Default spacing is 2px.
