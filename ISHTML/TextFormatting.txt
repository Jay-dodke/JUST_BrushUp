Text formatting tags -> change the appearance or emphasis of text
<b> -> bold (visual)
<strong> -> strong importance (semantic)
<i> -> italic (visual)
<em> -> emphasized (semantic)
<u> -> underline
<mark> -> highlight
<small> -> smaller text
<del> -> deleted text
<ins> -> inserted text
Example -> <p><strong>Important</strong> and <em>emphasized</em> text.</p>
