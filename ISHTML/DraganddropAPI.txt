HTML Drag and Drop API

What it is
- Drag and Drop allows users to drag elements and drop them into another area.
- HTML supports drag and drop with JavaScript events.

Key events
- `ondragstart` - when dragging starts
- `ondragover` - when the dragged item is over a target
- `ondrop` - when the item is dropped

Best example
```
<div id="div1" ondrop="drop(event)" ondragover="allowDrop(event)"></div>
<img id="drag1" draggable="true" ondragstart="drag(event)">

<script>
function allowDrop(event) {
  event.preventDefault();
}
function drag(event) {
  event.dataTransfer.setData("text", event.target.id);
}
function drop(event) {
  event.preventDefault();
  const data = event.dataTransfer.getData("text");
  event.target.appendChild(document.getElementById(data));
}
</script>
```

Notes
- You must call `event.preventDefault()` in `ondragover` to allow dropping.
- The `dataTransfer` object carries the dragged data.
