HTML Layout

What it is
- A layout describes how content is arranged on a web page.
- HTML5 provides semantic elements for layout structure.

Common layout elements
- `<header>` - top area or page title
- `<nav>` - navigation links
- `<section>` - grouped content
- `<article>` - main content block
- `<aside>` - side content
- `<footer>` - bottom area

Simple layout example
```
<header>
  <h1>My Website</h1>
</header>

<section>
  <nav>
    <ul>
      <li>Home</li>
      <li>News</li>
      <li>Contact</li>
    </ul>
  </nav>

  <article>
    <h2>Main Content</h2>
    <p>Page details go here...</p>
  </article>
</section>

<footer>
  <p>Footer area</p>
</footer>
```

Notes
- Use CSS to control sizes, colors, and positioning.
- Responsive layouts adjust for small screens with media queries.
