HTML Table Size

What it is
- Table size is controlled with CSS.
- You can set width/height for the table or for specific cells.

Set table width
```
<table style="width: 100%;">
  ...
</table>
```

Set column width
```
<style>
  th, td { width: 150px; }
</style>
```

Set row height
```
<style>
  tr { height: 40px; }
</style>
```

Best practice
- Use percentages for flexible layouts.
- Set sizes on `th`/`td` when you want consistent column widths.
