HTML Script Tag

What it is
- The `<script>` tag defines a client-side script (JavaScript).
- A `<script>` element can contain code or use the `src` attribute to link an external script file.
- JavaScript is used for dynamic and interactive behavior.

Basic example
```
<script>
  document.getElementById("demo").innerHTML = "Hello JavaScript!";
</script>
```

External script
```
<script src="myscript.js"></script>
```

Noscript
```
<noscript>Sorry, your browser does not support JavaScript!</noscript>
```

Notes
- Put scripts in `<head>` or before `</body>`.
- `document.getElementById()` is commonly used to select elements by id.
