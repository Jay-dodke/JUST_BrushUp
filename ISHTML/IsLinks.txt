HTML Links (Hyperlinks)

What it is
- HTML links are created with the `<a>` tag.
- The `href` attribute sets the link destination.
- Link text is what users click.

Basic syntax
```
<a href="url">link text</a>
```

Common examples
```
<a href="https://www.example.com">Visit Example</a>
<a href="about.html">About (relative link)</a>
<a href="mailto:someone@example.com">Send email</a>
```

Target attribute
- `target="_self"` opens in the same tab (default).
- `target="_blank"` opens in a new tab.

Best practice
- Use clear, meaningful link text.
- Prefer relative URLs for internal pages and absolute URLs for external sites.
- Style links with CSS if you want a custom look.
