HTML Computer Code Elements

What it is
- HTML provides tags for displaying computer code and related text.

Elements
- `<code>` - inline code
- `<pre>` - preformatted text (keeps spaces and line breaks)
- `<kbd>` - keyboard input
- `<samp>` - sample output from a program
- `<var>` - variables

Examples

Inline code
```
<p>Use <code>let x = 5;</code> in JavaScript.</p>
```

Preformatted code
```
<pre><code>
function greet(name) {
  return "Hello " + name;
}
</code></pre>
```

Keyboard input
```
Press <kbd>Ctrl</kbd> + <kbd>S</kbd> to save.
```

Sample output
```
Program output: <samp>Hello World!</samp>
```

Variable
```
The value of <var>x</var> is 10.
```

Notes
- Use `<pre>` with `<code>` when showing multi-line code.
- These tags add meaning for browsers and assistive tech.
