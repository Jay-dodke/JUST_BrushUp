HTML Table Styling

What it is
- Table styling is done with CSS to improve readability.
- Common styles include zebra stripes, hover effects, and dividers.

Zebra stripes (rows)
```
tr:nth-child(even) {
  background-color: #D6EEEE;
}
```

Vertical zebra stripes (columns)
```
td:nth-child(even), th:nth-child(even) {
  background-color: #D6EEEE;
}
```

Horizontal dividers
```
tr {
  border-bottom: 1px solid #ddd;
}
```

Hoverable rows
```
tr:hover { background-color: #D6EEEE; }
```

Best practice
- Use subtle colors for stripes and hover to keep text readable.
