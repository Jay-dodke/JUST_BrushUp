HTML Buttons

What it is
- The `<button>` element defines a clickable button.
- A button does nothing by itself until you add an action (like JavaScript or a form submit).

Simple example
```
<button>Click Me</button>
```

Style a button with CSS
```
<button class="btn">Green Button</button>

<style>
  .btn {
    background-color: #2e7d32;
    color: white;
    padding: 10px 16px;
    border: none;
    border-radius: 6px;
  }
</style>
```

Disabled button
```
<button disabled>Disabled Button</button>
```

Button with JavaScript
```
<button onclick="alert('Hello!')">Click Me</button>
```

Button types
- `type="button"` - normal button (no default action)
- `type="submit"` - submits a form
- `type="reset"` - resets form fields

Form example
```
<form>
  First name: <input type="text" name="fname">
  <button type="submit">Submit</button>
  <button type="reset">Reset Form</button>
</form>
```

Notes
- Always specify the `type` attribute.
- Inside a form, the default button type is `submit` if you omit it.
