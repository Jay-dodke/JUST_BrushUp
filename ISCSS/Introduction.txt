CSS Introduction

What is CSS?
- CSS stands for Cascading Style Sheets.
- CSS describes how HTML elements are displayed on screen, paper, or in other media.
- CSS can control the layout of multiple web pages at once.
- External stylesheets are stored in .css files.

Why use CSS?
- It separates content (HTML) from design (CSS).
- It enables different styles for different devices and screen sizes.

CSS saves work
- Changing a single external stylesheet can update the look of an entire site.

Example CSS
```
body {
  background-color: lightblue;
}

h1 {
  color: white;
  text-align: center;
}

p {
  font-family: verdana;
  font-size: 20px;
}
```
