CSS Padding and Box Sizing

By default, padding adds to an element's total width. The box-sizing property lets you control this behavior.

Default (content-box)

Width applies to the content only. Padding is added to the total size.

width: 300px; padding: 25px;

With box-sizing: border-box

Padding and border are included in the width and height.

width: 300px; padding: 25px; box-sizing: border-box;

Apply to All Elements

Many developers set box-sizing: border-box; globally. 

* {
  box-sizing: border-box;
}
