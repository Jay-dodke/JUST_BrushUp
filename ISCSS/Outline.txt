CSS Outline

What it is
- An outline is a line drawn outside the element's border.
- It does not take up space, so it won't affect layout.

Outline properties
- outline-style: dotted, dashed, solid, double, groove, ridge, inset, outset, none, hidden.
- outline-width: thin, medium, thick, or a length (px, etc.).
- outline-color: any valid color.
- outline-offset: space between border and outline.

Shorthand
- outline: width style color;

Examples
```
.box { outline-style: solid; outline-color: red; }
.box { outline-width: 5px; }
.box { outline: 4px solid #2196f3; }
.box { outline-offset: 10px; }
```
