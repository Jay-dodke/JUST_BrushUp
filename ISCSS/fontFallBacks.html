CSS Font Fallbacks

A font stack lists preferred fonts first and ends with a generic family so the browser can choose the best available option.

Why Fallbacks Matter

Not every device has the same fonts installed. A fallback stack ensures readable text and a consistent feel, even when the primary font is missing.

Serif Stack

"Times New Roman", Times, serif

Serif fonts have small strokes at the ends of letters and feel traditional.

Sans-Serif Stack

"Segoe UI", Tahoma, Arial, sans-serif

Sans-serif fonts are clean and modern, ideal for screens.

Monospace Stack

"Consolas", "Courier New", monospace

Monospace fonts align characters evenly and are common in code samples.

How to Write a Stack

  • List your preferred font first.
  • Add similar fonts as backups.
  • End with a generic family (serif, sans-serif, monospace).

Professional Example

body {
  font-family: "Segoe UI", Tahoma, Arial, sans-serif;
}

h1 {
  font-family: "Georgia", "Times New Roman", serif;
}
