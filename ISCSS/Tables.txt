CSS Tables

Overview
- Table styling commonly uses border, border-collapse, padding, and border-spacing.
- Apply borders to table, th, and td for visible grid lines.
- Use border-collapse to avoid double borders.

Table borders
- border is shorthand for border-width, border-style, and border-color.
- Apply to table, th, and td for cell borders.
- Apply only to table for an outside border.

Border collapse
- separate (default): each cell draws its own borders.
- collapse: borders collapse into a single border.

Padding and spacing
- padding on th and td adds space between content and borders.
- border-spacing controls space between adjacent cell borders (only with separate).

Example
```
table, th, td { border: 1px solid #333; }
table { border-collapse: collapse; }
th, td { padding: 10px 14px; }
table { border-collapse: separate; border-spacing: 12px; }
table { border: 1px solid #333; }
```
