CSS Text Color

What it does
- The color property sets the text color.
- Colors can be specified by name, HEX, or RGB values.
- The default text color is usually set in the body selector.

Text color and background color
- You can set both background-color and color together.
- Keep strong contrast between text and background for readability.

Example
```
body {
  color: blue;
}

h1 {
  color: green;
}

h2 {
  color: red;
}

body {
  background-color: lightgrey;
  color: blue;
}

h1 {
  background-color: black;
  color: white;
}

div {
  background-color: blue;
  color: white;
}
```
