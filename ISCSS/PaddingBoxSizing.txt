CSS Padding and Box Sizing

Key idea
- The CSS width property sets the width of the content area by default.
- Padding is added outside the content area, increasing the total size.

Default behavior (content-box)
- width and height apply only to content.
- Total size = width/height + padding + border.

box-sizing: border-box
- Width and height include content, padding, and border.
- Useful when you want a fixed overall width even with padding.

Common pattern
- Many developers set this globally so layouts are predictable:

```
* {
  box-sizing: border-box;
}
```

Example
```
.box {
  width: 300px;
  padding: 25px;
}

.box.border-box {
  box-sizing: border-box;
}
```
