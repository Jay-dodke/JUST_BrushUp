CSS Links

What you can style
- Links can be styled like normal text and also have special states.
- Common states: :link, :visited, :hover, :active.

State order (LVHA)
- :link ? :visited ? :hover ? :active
- This order prevents hover/active rules from being overridden.

Common patterns
- Remove underline with text-decoration: none.
- Add background and padding to make a link look like a button.

Example
```
a:link { color: #1a0dab; }
a:visited { color: #660099; }
a:hover { color: #d32f2f; text-decoration: underline; }
a:active { color: #0d47a1; }

.button-link {
  display: inline-block;
  padding: 10px 16px;
  background: #2b4c7e;
  color: #fff;
  text-decoration: none;
}
```
