CSS Margin Collapse

What it is
- When two vertical margins meet, they can combine into a single margin.
- Only vertical margins collapse (top/bottom). Horizontal margins do not.

Common cases
1. Adjacent siblings
   - The larger of the two margins is used.

2. Parent and first/last child
   - If the parent has no border, padding, or inline content, the child’s
     margin can collapse with the parent’s margin.

3. Empty elements
   - An empty element’s top and bottom margins can collapse together and with
     adjacent margins.

How to prevent collapsing
- Add padding or a border to the parent.
- Add inline content between elements.
- Create a new block formatting context (e.g., overflow: auto).

Example
```
.box-a { margin-bottom: 40px; }
.box-b { margin-top: 25px; }

.parent { border: 2px solid #c44; padding: 6px; }
.child { margin-top: 30px; }
```
