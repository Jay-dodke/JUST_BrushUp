CSS Text Alignment

Text-align
- text-align sets the horizontal alignment of text.
- Common values: left, right, center, justify.

Text-align-last
- text-align-last sets the alignment of the last line of a block.

Text direction
- direction controls the text direction.
- unicode-bidi works with direction to override the bidi algorithm.

Vertical align
- vertical-align aligns inline or table-cell elements.

Examples
```
h1 { text-align: center; }

p { text-align: justify; }

p { text-align-last: right; }

p { direction: rtl; unicode-bidi: bidi-override; }

img { vertical-align: middle; }
```
