CSS RGB and RGBA Colors

RGB colors are defined by red, green, and blue light values from 0 to 255. RGBA adds an alpha value for opacity.

RGB Value

rgb(red, green, blue)

Black is rgb(0, 0, 0) and white is rgb(255, 255, 255).

rgb(255, 0, 0)
rgb(0, 0, 255)
rgb(60, 179, 113)
rgb(238, 130, 238)
rgb(255, 165, 0)
rgb(106, 90, 205)

Shades of Gray

Equal RGB values create gray shades.

rgb(60, 60, 60)
rgb(90, 90, 90)
rgb(120, 120, 120)
rgb(180, 180, 180)
rgb(210, 210, 210)
rgb(240, 240, 240)

RGBA Value

rgba(red, green, blue, alpha)

Alpha ranges from 0.0 (transparent) to 1.0 (opaque).

rgba(255, 99, 71, 0)
rgba(255, 99, 71, 0.2)
rgba(255, 99, 71, 0.4)
rgba(255, 99, 71, 0.6)
rgba(255, 99, 71, 0.8)
rgba(255, 99, 71, 1)
