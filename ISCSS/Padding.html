CSS Padding

Padding creates space inside an element, between the content and the border.

Padding Example

This element uses padding: 70px.

Individual Sides

Top, right, bottom, and left can be set separately.

Different padding on each side.

Shorthand Examples

padding: 25px 50px 75px 100px;
padding: 25px 50px 75px;
padding: 25px 50px;
padding: 25px;

Valid Values

  • length (px, pt, cm, etc.)
  • % of the containing block width
  • inherit from the parent
  • Negative values are not allowed
