CSS Text Decoration Styles

What it does
- text-decoration-style sets the style of the decoration line.
- It works with text-decoration-line (underline, overline, line-through).

Values
- solid: single line (default).
- double: two parallel lines.
- dotted: dotted line.
- dashed: dashed line.
- wavy: wavy line.

Notes
- You can combine line, color, style, and thickness using text-decoration.

Example
```
h1 {
  text-decoration-line: underline;
  text-decoration-style: wavy;
}

h2 {
  text-decoration: underline red double 4px;
}
```
