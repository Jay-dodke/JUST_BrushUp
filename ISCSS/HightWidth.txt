CSS Height and Width

What they do
- The height and width properties set the size of the content area.
- They do not include padding, borders, or margins.

Common values
- length: px, cm, etc.
- %: percentage of the containing block.
- auto: browser calculates the size (default for height).

Examples
```
.box {
  width: 200px;
  height: 100px;
}

.box {
  width: 50%;
  height: 60px;
}

.box {
  width: 300px;
  height: auto;
}

.box {
  width: 100%;
  max-width: 500px;
}
```

Tip
- Use max-width to prevent wide elements on large screens while allowing
  them to shrink on small screens.
