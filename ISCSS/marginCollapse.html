CSS Margin Collapse

Vertical margins of block elements can combine into a single margin.

Adjacent Siblings

Two vertical margins collapse to the larger value.

Box A (margin-bottom: 40px)
Box B (margin-top: 25px)

Parent and First/Last Child

If the parent has no border/padding, the child`s top margin can collapse with the parent.

Child margin-top: 30px (collapses)

Child margin-top: 30px (no collapse)

Empty Blocks

An empty elements top and bottom margins can collapse with adjacent elements.

Box Above
(empty block)
Box Below

How to Prevent Collapsing

  • Add padding or a border to the parent.
  • Add inline content between elements.
  • Create a new block formatting context (e.g., overflow: auto).
