CSS Icons

Icons can be added with icon fonts (like Font Awesome) or built-in frameworks. Below are examples of how icon classes are used.

Font Awesome (class-based)

Example markup (icons require the Font Awesome library).

<i class="fa fa-cloud"></i> <i class="fa fa-heart"></i> <i class="fa fa-car"></i> <i class="fa fa-file"></i> <i class="fa fa-bars"></i>
fa fa-cloud fa fa-heart fa fa-car fa fa-file fa fa-bars

Bootstrap Icons (class-based)

Example markup (icons require Bootstrap Icons).

<i class="bi bi-alarm"></i> <i class="bi bi-emoji-smile"></i> <i class="bi bi-lightning"></i>
bi bi-alarm bi bi-emoji-smile bi bi-lightning

Google Material Icons

Example markup (icons require Material Icons).

<i class="material-icons">cloud</i> <i class="material-icons">favorite</i> <i class="material-icons">directions_car</i>
material-icons cloud material-icons favorite material-icons directions_car
