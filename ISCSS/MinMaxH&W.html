CSS Min/Max Width and Height

Use min-width, max-width, min-height, and max-height to control size limits.

Min-Width

The element will never be smaller than 200px.

min-width: 200px; width: 30%;

Max-Width

The element will never be wider than 500px.

max-width: 500px; width: 100%;

Min-Height

The element will never be shorter than 100px.

min-height: 100px;

Max-Height

Content will overflow if it exceeds the max height.

max-height: 120px; overflow: auto;
