CSS Icons

Overview
- Icons are often added with icon fonts (Font Awesome, Bootstrap Icons, Material Icons).
- You include the icon library in the page, then use class names or icon text.

Common approaches
1. Font Awesome
   - Use classes like fa fa-cloud.


2. Bootstrap Icons
   - Use classes like bi bi-alarm.

3. Google Material Icons
   - Use the material-icons class with icon names inside the tag.

Sizing
- Icon size is controlled with font-size.

Example
```
<i class="fa fa-cloud"></i>
<i class="bi bi-alarm"></i>
<i class="material-icons">favorite</i>

.big-icon {
  font-size: 48px;
}
```
